# AirIQ - Urban Air Quality Intelligence

This repository contains the full AirIQ web application and local server.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and update any required values:
   `cp .env.example .env`
3. Run the app:
   `npm run dev`

## Configuration

Update `.env` with the hosting URL and any external API keys used by the application.
